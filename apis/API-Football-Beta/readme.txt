Documentacion: https://rapidapi.com/api-sports/api/api-football-beta

Ejemplo de Informacion de las Ligas: https://api-football-beta.p.rapidapi.com/leagues?country=France Retorna un Array con las Ligas en Francia.
Ejemplo de Informacion de los Partidos: https://api-football-beta.p.rapidapi.com/fixtures?league=39&season=2020 Retorna un Array con los Partidos de la Liga 39 en la Temporada 2020.
Ejemplo de Informacion de las Predicciones: https://api-football-beta.p.rapidapi.com/predictions?fixture=fixtureId Retorna un Array con las Predicciones del Dia para ese Fixture
